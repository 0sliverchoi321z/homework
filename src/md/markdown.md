# 마크다운 문법

> **Mission-01**
> 마크다운 문법에 대해 조사하고 정리한다.

## 마크다운(Markdown)이란 무엇인가

마크다운이란 마크업 언어(Markup Languages)¹의 한 종류로, **일반 텍스트에 간단한 기호를 사용하여 문서의 구조와 서식을 표시하는 가벼운 문서 작성 언어**다.² 다시 말해, **읽기 쉬운 텍스트로 서식을 표현하는 문서용 문법**을 마크다운 문법이라고 한다.

이는 2004년 존 그루버에 의해 만들어졌으며, 마크다운 서식 문법의 궁극적인 설계 목표는 가능한 한 가독성을 높이는 것에 있다. 이상적인 마크다운 문서는 그대로 일반 텍스트로 공개(publish)해도 무리가 없고, 복잡한 서식 지시어로 지저분해 보이지 않아야 한다.

## 마크다운 문법의 예시

### 제목(Headings) `#`

'#'를 문장의 앞에 붙이는 것으로 제목의 크기를 변경할 수 있다. (1~6까지)
호환성을 유지하고자 한다면 문장 앞의 기호와 문자 텍스트 사이에 공백을 넣는 것에 유의.

```
# 제목1
## 제목2
### 제목3
#### 제목4
##### 제목5
###### 제목6
```

# 제목1

## 제목2

### 제목3

#### 제목4

##### 제목5

###### 제목6

#### 대체 구문 (Alternate Syntax) `==, --`

본문 아래에 '=='를 원하는 만큼 추가하는 것으로도 변경이 가능하다.
==를 붙이는 것은 #를 하나 붙인 것과, --를 붙이는 것은 #를 두 개 붙인 것과 동일하게 출력된다.

```
예시는 다음과 같습니다.
==
이것도 이렇게 됩니다.
--
```

# 예시는 다음과 같습니다.

## 이것도 이렇게 됩니다.

## 줄 바꿈 (Paragraphs)

빈 줄을 추가하거나, 문장 끝에 두 칸 이상의 공백을 입력하는 것으로 줄 바꿈을 할 수 있다.

## 강조 (Emphasis)

텍스트를 굵게, 또는 기울어진 이텔릭체로 표시하여 강조할 수 있다.

### 굵게 `*, _`

```
**강조**
__강조__
```

**강조**
**강조**

문장 중간의 단어를 강조할 때는 '\*'를 사용하자.

### 기울임 `**, __`

굵게 강조하기 위해 사용한 기호를 똑같이 활용한다.
단, 이번에는 하나씩만 붙여 사용한다.

```
*강조*
_강조_
```

_강조_
_강조_

### 굵게+기울임 `***, ___`

```
***강조***
___강조___
```

**_강조_**
**_강조_**

## 인용구 (Emphasis) `>`

단락 앞에 '>'를 붙인다.

```
> 인용구
```

> 인용구

인용구를 중첩하여 사용하고 싶을 경우에는 '>>'로 입력.

```
> 인용구
>> 인용구
```

> 인용구
>
> > 인용구

## 목록 (List)

### 순서가 있는 목록 `1, 2, 3...`

1. 순서가 없는 목록
   1-1. 들여쓰기 된 순서가 없는 목록
2. 순서가 없는 목록
3. 순서가 없는 목록

### 순서가 없는 목록 `-, *, +`

- 순서가 없는 목록
  - 들여쓰기 된 순서가 없는 목록

* 순서가 없는 목록

- 순서가 없는 목록
